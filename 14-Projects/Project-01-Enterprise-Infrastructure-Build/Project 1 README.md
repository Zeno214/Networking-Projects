# Project 01 - Enterprise Infrastructure Build

## Objective

Design and build the physical foundation for an enterprise networking and infrastructure homelab.

---

# Business Scenario

As the Infrastructure Engineer for a fictional company, my objective is to build the core infrastructure that will support future enterprise networking, virtualization, Windows Server services, wireless networking, monitoring, and automation.
